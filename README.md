# FoodieHub

周边食品店铺点评平台


平台核心实现内容：用户上传自己想要分享的店铺（如果店铺没有被上传的话），填写店铺相关的信息，在问答区或评论区进行互动，相互进行交流。

核心群体：想要完成信息流通的小范围内部人员，此平台能够保证信息真实性；且由于店铺由用户自主上传，故所有出现的店铺都是有被分享价值的。

功能清单：
1、店铺相关，上传、查看、交互（评分,问答,评论,菜单,图片）
2、收藏夹，能够收藏店铺，并在收藏夹中展示，点击跳转详细页
3、个人主页相关，能够展示个人信息，并展示最近动态（在哪里进行了交互）
4、消息页，接受公告与回复提醒
5、管理员相关，审核提交的店铺修改申请，用户与店铺管理
