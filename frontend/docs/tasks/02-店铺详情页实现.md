## 一、任务概览

基于现有项目骨架（主页已完工），实现完整的店铺详情页。新增约 **20 个组件**、**3 个 API 模块**，覆盖轮播图 → 评分交互 → 评论区（常驻发布框+回复展开）→ 问答区 → 菜单（横向滚动+详情弹窗）→ 店铺相册（3行网格+Lightbox）。

**布局**（两行）：

```

```

**评分交互逻辑**：

```
用户点击第 N 颗星
  ├─ 游客 → onLoginPrompt()
  └─ 已登录 → userRating === null
       ├─ 否（已评过） → 显示 "你的评分：★★★" 只读
       └─ 是 → 星星高亮到 N → Modal "确认评为 N 星？"
            ├─ 取消 → 恢复高亮前状态
            └─ 确认 → onRate(N) → Toast "评分成功"
                       → 更新显示为 "你的评分：★★★★★" 只读
```

**评分分布**：
- 横向：`flex items-center gap-3`
- 左侧：⭐ 4.3 大号（`text-4xl font-bold`）+ 小数
- 中间：5 行分布条（5→1 星从上到下），每行：`flex items-center gap-2`
  - 标签 "5星" （`text-xs text-gray-400 w-6`）
  - 条形：`h-2 rounded-full bg-gray-200`，内嵌 `bg-orange-400` 按百分比填充宽度
  - 纯 CSS，不显示数字
- 底部："共 32 人评分"（`text-sm text-gray-400`）
- 无评分时：评分显示"--"，分布条全灰，文字 "暂无评分"
