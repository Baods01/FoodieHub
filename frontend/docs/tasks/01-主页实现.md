# 01 — 主页实现执行书

> **依据**：[specs/01-主页设计.md](../specs/01-主页设计.md)

## 任务概览

实现主页全部 6 个区域：导航栏 → 公告横幅 → 搜索框 → 排序/筛选 → 店铺卡片网格 → 边界状态。响应式 4/2/1 列，mock 数据驱动，API 层预留后端对接接口。

## Agent 拆分

```
Agent 1 (infra)       → types/ + api/ + hooks/ + store/
Agent 2 (layout)      → Header + RootLayout + AnnouncementBanner
Agent 3 (search)      → SearchBar + SortFilterBar + FilterDropdown
Agent 4 (shopcard)    → ShopCard + ShopList + Skeleton + EmptyState + ErrorState + useInfiniteScroll
Agent 5 (integration) → HomePage.tsx + router + main.tsx + index.css
```
