# 任务执行书

> 该目录存放面向 Claude Code 的多 Agent 并行执行任务书。
> 每份任务书包含：任务概览 → Agent 拆分 → 逐个文件实现规格 → 合并策略 → 验证清单。

## 执行规则

1. **按优先级执行**：编号越小越优先
2. **依赖检查**：每个任务开头注明前置条件
3. **Agent 并行**：独立 worktree 分别实现，主 Agent 协调合并
4. **mock 优先**：后端接口未就绪时先用 mock 数据，接口签名以注释保留
