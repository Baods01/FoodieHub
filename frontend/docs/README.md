## 文档结构

```
docs/
├── README.md              ← 当前文件：总入口 + 关键词检索
├── 01-项目理解.md           ← 项目背景、用户角色、功能清单
├── 02-技术方案.md           ← 技术选型、依赖安装状态、理由
├── 03-开发路线图.md          ← 目录结构、阶段进度、后端接口清单
├── specs/                  ← 页面/组件设计规格（实现依据）
│   ├── README.md
│   └── 01-主页设计.md
└── tasks/                  ← 可执行的实现任务书
    ├── README.md
    └── 01-主页实现.md
```
