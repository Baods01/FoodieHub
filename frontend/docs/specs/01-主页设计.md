# 主页设计规格

## 页面定位

主页是一个轻量化的搜索筛选工具入口。用户通常是其他平台看到某家店、带着目的来查评价。页面核心功能是帮助用户在顶部快速输入/筛选 → 在下方快速浏览结果 → 点击进入详情。

## 页面布局（从上到下）

```

├─────────────────────────────────┤
│      搜索框                     │
├─────────────────────────────────┤

├─────────────────────────────────┤
│  ┌────────┐  ┌────────┐        │
│  │ 店铺卡片 │  │ 店铺卡片 │        │

```

## 逐区域规格

### 1. 顶部导航栏

| 项目 | 规格 |
|:---|:---|
| 左侧 | "食探社"文字 Logo，点击跳转首页 |
| 右侧 | 已登录：圆形头像 32px + 昵称 → 跳转个人中心；未登录："登录/注册"文字按钮 |
| 背景 | 白色，底部 1px 浅灰分割线 |
| 高度 | 56px（h-14） |

### 2. 公告/图片横幅区

| 项目 | 规格 |
|:---|:---|
| 高度 | 固定 120px，垂直居中 |
| 有公告 | 浅橙 #FFF7F0 背景，橙色左边框，标题+正文摘要，line-clamp-2 |
| 无公告 | 灰色占位区 + "发现华农周边美食" 文字 |

### 3. 搜索框

| 项目 | 规格 |
|:---|:---|
| 位置 | 横幅下方，max-w-[600px] 居中 |
| 占位 | "搜索店铺名称..." |
| 行为 | 300ms 防抖自动触发，不清除不清空 |
| 清除 | 有内容时右侧出现 × 按钮 |
| 搜索历史 | 焦点时下拉最近 5 条，点击直接搜索 |

### 4. 排序与筛选栏

| 项目 | 规格 |
|:---|:---|
| 布局 | flex justify-between, 一行 |
| 排序 | Select 下拉：收藏最多(默认)、浏览量最高、最高评分、最多评论 |
| 品类筛选 | Listbox 下拉：「全部」+ 动态品类列表 |
| 区域筛选 | Listbox 下拉：「全部」+ 动态区域列表 |
| 清除 | 右侧"清除"按钮重置全部条件 |

### 5. 店铺卡片列表

| 项目 | 规格 |
|:---|:---|
| 布局 | CSS Grid: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4` |
| 分页 | IntersectionObserver 哨兵 + 200px rootMargin，每页 12 张 |
| 封面 | 16:9 aspect-ratio，有图 object-cover，无图灰色占位 + "暂无图片" |
| 名称 | text-base font-bold truncate |
| 标签 | 品类：橙色边框；区域：灰色边框 |
| 评分行 | ⭐ 4.3（灰字）+ 👁 320（灰字），flex justify-between |
| 卡片 | white rounded-lg shadow，hover 时 shadow-lg -translate-y-0.5 transition 200ms |
| 点击 | 整张卡片 `navigate(/shop/:id)` |

### 6. 边界状态

| 状态 | 表现 |
|:---|:---|
| 首次加载 | 8 张骨架屏（灰色脉冲动画，模仿卡片形状） |
| 搜索无结果 | 居中 SVG 空盒图标 + "还没有店铺" + "成为第一个分享的人" 按钮 |
| 网络失败 | 警示图标 + "加载失败" + "点击重试" 按钮 |

## 风格规范

| 维度 | 值 |
|:---|:---|
| 背景主色 | #FFFFFF |
| 品牌高亮色 | 渐变 #FF7E3A → #FF9A5C |
| 卡片圆角 | 8px |
| 头像 | 圆形 border-radius: 50% |
| 间距 | 卡片 gap-4，内容区 p-3 |
| 字体 | 系统默认中文，正文 14px |

## 响应式断点

| 视口 | 卡片列数 |
|:---|:---|
| > 1024px | 4 列 |
| 641-1024px | 2 列 |
| ≤ 640px | 1 列 |

## 游客 vs 登录用户

仅导航栏右侧差异：游客显示"登录/注册"，登录用户显示头像+昵称。其余区域完全一致。

## 技术实现约束

- **组件拆分**：Header, AnnouncementBanner, SearchBar, SortFilterBar, FilterDropdown, ShopCard, ShopList, ShopCardSkeleton, EmptyState, ErrorState — 各自独立
- **筛选状态**：useReducer 管理 `{ keyword, sort, category, area, page }`
- **防抖**：useDebounce 300ms
- **无限滚动**：IntersectionObserver
- **Mock 数据**：20 条覆盖 9 品类 × 5 区域 × 评分 2.0-5.0
