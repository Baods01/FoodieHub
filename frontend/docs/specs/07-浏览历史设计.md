# 07 — 浏览历史设计规格

> 用户浏览过的店铺记录页。横版卡片布局，与收藏夹保持一致的视觉语言。

---

## 页面定位

用户从 Header 下拉菜单 →「浏览历史」到达。记录用户曾经访问过的店铺，按浏览时间倒序排列，方便回溯之前看过的店。

---

## 逐区域规格

### 1. 顶部标题栏

| 项目 | 规格 |
|:---|:---|
| 左侧 | 🕐 `Clock` 图标（`text-blue-500`）+「浏览历史」`text-xl font-bold` |
| 右侧 | [清空记录] 按钮（`text-sm text-gray-400 hover:text-red-500`）+「共 N 条」`text-sm text-gray-400` |

### 2. 横版卡片布局

与收藏夹「FavoriteCard」保持一致的设计：

```
┌────────────┬─────────────────────────────────┐
│            │  店铺名称（text-base font-bold）  │
│   封面图   │                                  │
│   w-40     │  品类标签 · 地区标签（flex gap-2）│
│   h-28     │  N条评论 · N次浏览               │
│            │  text-sm text-gray-400           │
│   object-  │                                  │
│   cover    │  [移除]          浏览于 3小时前    │
└────────────┴─────────────────────────────────┘
```

| 项目 | 规格 |
|:---|:---|
| 整体 | `bg-white rounded-xl shadow-sm border border-gray-100`，flex row，`overflow-hidden` |
| 点击 | 整张卡片 `cursor-pointer`，跳转 `/shop/:id` |
| 悬停 | `hover:shadow-md hover:-translate-y-0.5 transition-all`（与收藏夹一致） |

### 3. 图片区（左侧）

| 项目 | 规格 |
|:---|:---|
| 尺寸 | `w-40 h-28 flex-shrink-0` |
| 无图态 | 灰色 `bg-gray-200` + 居中「暂无图片」 |

### 4. 信息区（右侧）

| 区域 | 内容 | 规格 |
|:---|:---|:---|
| 右上角 | ⭐ 评分 | `text-sm text-gray-400`，⭐ + 数字 |
| 左上 | 店铺名称 | `text-base font-bold truncate` |
| 第二行 | 品类标签 + 地区标签 | `flex gap-2`，品类橙，地区灰 |
| 第三行 | 冗余统计 | `text-sm text-gray-400`，「N条评论 · N次浏览」|
| 左下 | [移除] 按钮 | `text-sm text-gray-400 hover:text-red-500 transition-colors` |
| 右下 | 浏览时间 | `text-xs text-gray-400`，「浏览于 X小时前/X天前」|

### 5. 移除交互

| 操作 | 行为 |
|:---|:---|
| 点击单条「移除」 | 乐观更新：立即从列表中移除该条，调用 `historyApi.removeHistory(shopId)` |
| 移除失败 | 回滚出现 + Toast「操作失败，请重试」 |
| 点击「清空记录」 | 弹出确认 Dialog：「确定清空所有浏览记录？」+ [取消]/[确认] |
| 清空确认 | 乐观清空列表，调用 `historyApi.clearHistory()` |
| 清空失败 | 回滚出现 + Toast「操作失败，请重试」 |

### 6. 数据去重

| 规则 | 说明 |
|:---|:---|
| 同一店铺多次访问 | 列表中只保留最近一次访问记录，仅更新 `viewedAt` 时间 |
| 排序 | 按浏览时间倒序（最新的在最前） |
| 排序切换 | 不做，固定按时间倒序 |

### 7. 边界状态

| 状态 | 处理 |
|:---|:---|
| 加载中 | 3 行横版骨架屏（左侧 w-40 h-28 灰块 + 右侧文字灰条） |
| 空历史 | 🕐 `Clock` 大图标（`text-gray-200`）+「暂无浏览记录」+「去看看 →」链接首页 |
| 加载失败 | `ErrorState` 重试按钮 |
| 清空确认 | `Dialog`弹窗：「确定清空所有浏览记录？」两个按钮 |

---

## 数据流

```
页面加载
  ↓
historyApi.fetchHistory()
  ↓ 成功 → 渲染列表（不重复店铺，按 viewedAt 倒序）
  ↓ 失败 → ErrorState

点击「移除」
  ↓ 乐观移除 + 调 API
  ↓ 成功 → 保持移除
  ↓ 失败 → 回滚 + Toast

点击「清空记录」
  ↓ Dialog 确认
  ↓ 确认 → 乐观清空 + 调 API
  ↓ 失败 → 回滚 + Toast
```

---

## 接口依赖

| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | `/api/history` | 获取浏览历史（不重复店铺），返回 `HistoryItem[]` |
| DELETE | `/api/history/:shopId` | 移除单条浏览记录 |
| DELETE | `/api/history` | 清空全部浏览记录 |
