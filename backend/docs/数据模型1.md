# Users 表

| 字段名     | 类型         | 属性          | 说明                                        |
| ---------- | ------------ | ------------- | ------------------------------------------- |
| id         | INT          | 主键，自增    | 用户唯一标识                                |
| username   | VARCHAR(50)  | 非空，唯一    | 登录用户名                                  |
| password   | VARCHAR(255) | 非空          | 加密后的密码                                |
| phone      | VARCHAR(20)  | 非空          | 手机号                                      |
| email      | VARCHAR(100) | 非空          | 电子邮箱，用于通知                          |
| avatar     | VARCHAR(255) | 可选          | 头像图片URL                                 |
| bio        | TEXT         | 可选          | 个人简介                                    |
| role       | INT          | 非空，默认'0' | 角色：0:user（普通用户）、1:admin（管理员） |
| is_active  | BOOLEAN      | 默认TRUE      | 是否启用（软删除）                          |
| created_at | DATETIME     | 非空          | 注册时间                                    |
| updated_at | DATETIME     | 非空          | 最后更新时间                                |

# Shops 表

| 字段名         | 类型         | 属性                         | 说明                                         |
| -------------- | ------------ | ---------------------------- | -------------------------------------------- |
| id             | INT          | 主键，自增                   | 店铺唯一标识                                 |
| name           | VARCHAR(100) | 非空                         | 店铺名称                                     |
| description    | TEXT         | 可选                         | 店铺描述                                     |
| is_oncampus    | BOOLEAN      | 可选                         | 校内校外                                     |
| is_active      | BOOLEAN      | 默认TRUE                     | 是否启用（软删除）                           |
| created_at     | DATETIME     | 非空                         | 创建时间                                     |
| updated_at     | DATETIME     | 非空                         | 最后更新时间                                 |
| view_count     | INT          | 默认0                        | 总浏览量（冗余字段）                         |
| favorite_count | INT          | 默认0                        | 收藏数（冗余字段）                           |
| comment_count  | INT          | 默认0                        | 讨论数（冗余字段）                           |
| average_rating | DECIMAL(2,1) | 默认0.0                      | 平均评分（冗余字段）                         |
| category       | INT          | 外键，引用categories(id)     | 店铺类别，如“中餐”“咖啡”，可能需要关联表 |
| dining_methods | INT          | 外键，引用dining_methods(id) | 点餐方式，如"美团"“淘宝”，可能需要关联表   |

# Dining_methods 表

# Ratings 表

| 字段名     | 类型     | 属性                | 说明                             |
| ---------- | -------- | ------------------- | -------------------------------- |
| id         | INT      | 主键，自增          | 评分唯一标识                     |
| user_id    | INT      | 外键，引用users(id) | 评分用户                         |
| shop_id    | INT      | 外键，引用shops(id) | 被评店铺                         |
| score      | TINYINT  | 非空，范围1-5       | 评分值                           |
| is_active  | BOOLEAN  | 默认TRUE            | 是否启用（软删除）               |
| created_at | DATETIME | 非空                | 评分时间                         |
| updated_at | DATETIME | 非空                | 最后更新时间                     |
| 唯一约束   | -        | (user_id, shop_id)  | 保证一个用户对一个店铺只能评一次 |

# Comments 表

| 字段名      | 类型       | 属性                                   | 说明                                                     |
| ----------- | ---------- | -------------------------------------- | -------------------------------------------------------- |
| id          | BIGINT(20) | 主键，自增                             | 评论唯一标识                                             |
| shop_id     | INT(11)    | 非空                                   | 关联的店铺ID（或其它目标实体ID）                         |
| user_id     | INT(11)    | 非空，外键引用users(id)                | 发表评论的用户ID                                         |
| parent_id   | BIGINT(20) | 可为空，默认 NULL                      | 父评论ID。NULL 或 0 表示一级评论；非空表示对该评论的回复 |
| root_id     | BIGINT(20) | 可为空，默认 NULL                      | 根评论ID（所属的一级评论ID），用于快速聚合和分页         |
| content     | TEXT       | 非空                                   | 评论内容，支持富文本或纯文本                             |
| like_count  | INT(11)    | 非空，默认 0                           | 点赞数，冗余字段，便于排序和展示                         |
| reply_count | INT(11)    | 非空，默认 0                           | 直接子回复数量，冗余字段，减少 COUNT 查询                |
| is_active   | BOOLEAN    | 默认TRUE                               | 是否启用（软删除）                                       |
| created_at  | DATETIME   | 非空，默认 CURRENT_TIMESTAMP           | 创建时间                                                 |
| updated_at  | DATETIME   | 非空，默认 CURRENT_TIMESTAMP ON UPDATE | 最后更新时间（如内容修改、点赞数变化等）                 |

# Images 表

# Messages 表

# User_behavior_logs 表

| 字段名        | 类型        | 属性                        | 说明                                                                                    |
| ------------- | ----------- | --------------------------- | --------------------------------------------------------------------------------------- |
| id            | BIGINT      | 主键，自增                  | 日志唯一标识                                                                            |
| user_id       | INT         | 外键，引用users(id)，可为空 | 操作用户（未登录用户可为空）                                                            |
| session_id    | VARCHAR(64) | 可选                        | 会话ID，用于追踪未登录用户行为                                                          |
| behavior_type | VARCHAR(32) | 非空                        | 行为类型，如 view_shop、rate_shop、comment、favorite、share 等                          |
| target_type   | VARCHAR(32) | 非空                        | 操作目标类型，如 shop、menu_item、comment、question 等                                  |
| target_id     | INT         | 非空                        | 目标对象的ID                                                                            |
| ip_address    | VARCHAR(45) | 可选                        | 客户端IP地址                                                                            |
| user_agent    | TEXT        | 可选                        | 客户端设备信息                                                                          |
| is_active     | BOOLEAN     | 默认 TRUE                   | 软删除标志，FALSE 表示已删除但不物理移除                                                |
| created_at    | DATETIME    | 非空                        | 行为发生时间                                                                            |
| updated_at    | DATETIME    | 非空                        | 最后更新时间                                                                            |
| 索引建议      | -           | -                           | 针对 user_id、target_type+target_id、behavior_type、created_at 建立索引，以加速查询分析 |
